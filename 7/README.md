# Ruby Game

This is a Ruby-based game project that uses Minitest for testing.

## Prerequisites

- Ruby 2.7.5
- Bundler gem

## Setup

1. Install dependencies:
```bash
bundle install
```

## Running the Game

To start the game, run:
```bash
ruby app.rb
```

## Running Tests

The project uses Minitest for testing. To run the tests:

```bash
ruby test/all_tests.rb
```
